Read & Write Entries From the HostsFile with PHP

## Installation

You can install the package via composer:

```bash
composer require appstract/php-hostsfile
```

## Usage

```php
$host = new Appstract\HostsFile();
$lines = $host->getLines();
```

## Testing

```bash
$ composer test
```
